The timings have been refined. They represent averages among individual differences.
Individuals’ performance can be extended in time by practice. Moreover, the fall-off in par-

though small superiority can be discerned, on average, up to 500 ms after stimulus offset.829

It is not obvious whether perceptions retained in the memory are phenomenally con-
scious. Subjects report having seen all or almost all items. It seems likely that some general
aspects of the initial perceptions are phenomenally conscious. They are surely not rational-
access conscious.

To summarize the discussion so far: The sensory icon is retains the phenomenological
perceptual image. Iconic memory retains more than one row. The representational infor-
mation in iconic memory is not available to free report and is not rational-access con-
scious. It is available through more indirect means. The memory capacity with the smaller
capacity (4.5 items out of 12), whose contents and is available to full, free report, is (short-
term) working memory.

Many types of representational content are preserved in iconic memory: color, lightness,
direction of rotary or linear motion, orientation of gratings, flicker, spatial location. Iconic
memory is a perceptual memory—a memory that retains representational content from per-
ceptual states. It retains perceptual attributives for shapes, bodies, letters, nu merals, and so
on. It thus includes categorizing attributives.830 It seems obvious that neither visual sensory
memory nor visual iconic memory is supra-perceptual in the sense we have been discussing.

Fragile Visual Short-Term Memory

Recent evidence suggests that there is a memory system called fragile visual short-term
memory (FVSTM).831 It is not obvious that FVSTM is a distinct memory system. It bears
close relations both to iconic memory and to visual working memory (discussed below). It
may be an implementation of one or both of these systems. I discuss it as a system.
