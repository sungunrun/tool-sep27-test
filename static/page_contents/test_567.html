Perceptual Memory I: Shorter Term Systems

. . . I see the awakening of consciousness as a series of spaced flashes, with the

intervals between them gradually diminishing until bright blocks of percep-

tion are formed, affording memory a slippery hold.

Vladimir Nabokov, Speak, Memory

Memory, as I understand the term, is a competence that functions to encode and retain rep-
resentation for further use. As a matter of terminology, I do not count as memory those
types of retention that preserve non-representational information—“muscle memory” and
sub-perceptual sensory retention. I do not include a capacity for retrieval as constitutively
associated with memory. Some memories operate only at unconscious levels, showing them-
selves in behavior. Some, at least in lower animals, may be unretrievable into consciousness.

Every perceptual modality—including vision, hearing, touch, proprioception—comes
with two closely connected representational systems in the same modality: perceptual
memory and perceptual anticipation. For example, in every known psychology that has
visual perception, that capacity is coupled with at least some sort of short-term visual
memory and some sort of visual anticipation. Some perceptual systems come with a sys-
tem of perceptual imagining as well. Imagining may be absent in the psychologies of lower
animals that have perception.

Each of these generic representational functions—memory, anticipation, and imagin-
ing—has a species that is functionally associated with perception. Each makes use of, and is
parasitic on, a perceptual system in the same modality. In this and the next chapter, I discuss
visual perceptual memory. In Chapter 18, I discuss visual anticipation and imagination.

Main tasks in Chapters 16 and 17 are two. First, I survey perceptual-memory systems.
Second, I extend the point made about categorization, conation, and attention: Memory comes
in representational levels. Memory per se should not be regarded as cognition, where cognition
is understood as a competence at a supra-perceptual level of representation that is more
knowledge-like or more sophisticated than perception. Some memory is perceptual-level.

Various types of visual memory are perceptual-level. All function to retain visual percep-
tual attributives. Psychological operations involved in perceptual memory are in no way
more sophisticated or more knowledge-like than operations that form perceptions. The vari-
ous types of perceptual memory are part of the system of perceptual processing. Detailed
descriptions of different types of visual memory are partly intended to support these points.

Calling memory cognition has been encouraged by loose use of auxiliary terms. In the
scientific literature, memory is regularly called ‘knowledge’.807 The effect of this usage is to
over-intellectualize the psychological kind.
