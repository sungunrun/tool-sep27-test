It is not known whether the different types of attention are truly distinct systems. They
may be different aspects of a single system. There is some evidence that exogenous and
endogenous attention are supported by different mechanisms, while drawing on a single
capacity-limited system. Exogenous applications tend to be faster and applied earlier in
the processing sequence. Endogenous applications have a more flexible, potentially longer
application period.747

It has been claimed that attention does not affect the nature of perceptual states.748 The
implication is that it only enhances their acuity or efficiency. I think the claim mistaken.
Attention affects how, as well as how well, entities are perceived. This is true even when
attention improves acuity. In sharpening a fuzzy perception, or in enhancing contrast,
attention changes a mode of presentation—which is to say, it changes the state-type.
A sharper perception of the same entity can produce faster responses. Since the sharper
perception enters into different law-like causal patterns, it is a different perceptual-state
kind. It has a different representational content—a different mode of presentation. Greater
acuity also brings into view boundaries, gaps, contours not otherwise perceived. So not
only mode of presentation, but representata of perception are affected by attention. So
attention can change accuracy conditions. Mode of presentation, representational content,
the condition made accurate or inaccurate, psychological state (all these phrases specify
the same thing) change when attention affects a perceptual state.

Attention and Accuracy

Given that the visual system has a finite number of resources, greater allocation of pro-
cessing power to one operation usually yields a correspondingly lesser allocation to
another. It can be safely assumed that, in the main, that there are practical benefits from
initiations of attention. There are also usually representational benefits. Exogenous and
endogenous spatial attention speed processing, improve grouping, enhance luminance
contrast and other contrast sensitivities—making discriminations of visual objects and
properties easier—, augment position coding, and improve spatial resolution or acuity.
The initiation and effects of attention can occur either during perceptual processing, or via
a cue prior to processing.749
