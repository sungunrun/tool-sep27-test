I think that there may be perceptual kinds marked by early perceptual processing.
However, the view misses more fundamental perceptual and perceptual-level psy cho-
logic al kinds that extend further and are richer in content than the very first formations of
perception. These kinds include intra-saccadic processing up through enriched, detailed
categorization. Quite a lot of trans-saccadic processing is perceptual-level. After its earliest
stages in the first 150 ms, which already includes perceptual categorization, visual compu-
tational processing takes substantial input from conation, memory, anticipation, im agin-
ing, attention, learning, and affect. All this input is perceptual level. None of this
perceptual-level processing constitutively includes cognitive elements in any sense of
‘cognitive’ that counts cognition a higher-level type of representation than perception.
Computational explanations of perceptual-level processing at these late stages omit supra-
perceptual-level states, including propositional states, without inducing explanatory
failure.

The sense in which perceptual-level systems do not include cognitive elements is not
that their processing is not, sometimes, causally affected in normal ways by cognitive
elem ents. One should not think of psychological systems as causally hermetic. Chapters
14–18 develop the idea that each of the listed generic capacities has a perceptual level as
well as supra-perceptual, or cognitive, levels. For example, memory or attention com-
mands are not monoliths. They have representational levels. There is interaction between
levels. However, computations in perceptual psychology operate only on perceptual-level
representations. This fact grounds taking them to be part of a system, which does not
include supra-perceptual representation.

Opponents of cognitive encapsulation or impenetrability rightly insist that perceptual
processing is not as feed-forward, as the original proponents of encapsulation thought.
They rightly take non-perceptual input, from memory and attention, to contribute to per-
ceptual processes, such as some (but not all) instances of perceptual categorization.
Opponents have rightly criticized some of Fodor and Pylyshyn’s claims about impenetra-
bility, on natural understandings of the claims.

However, both proponents and opponents tend to think of memory, anticipation,
im agin ing, learning—and often attention initiations and conation—as monoliths. They
tend to think of some or all of them as cognitive, full stop. For example, it is common to
infer from perceptual processing as task oriented—influenced by conation—to its being
cognitively penetrated. Since I think that the perceptual-level versions of these generic
capacities occur inside the perceptual (or perceptual-motor) system, I reject thinking that
they could penetrate it in any sense.

Many on both sides of the penetration dispute show little reflection on what counts as
cognition—the candidate “penetrator”. This neglect is mainly a philosophical failure.
