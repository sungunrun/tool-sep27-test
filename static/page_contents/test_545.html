Sources and Levels of Attention

Attention functions to select information that is likely to be relevant to practical matters.
Certain stimulus types automatically attract exogenous attention, such as features that
pop-out and abrupt onsets such as sudden motions. Both exogenous and endogenous
attention are guided not only by present stimuli, but also by antecedent, practically rele-
vant resources—especially affect and memories of action targets. Exogenous and en dogen-
ous attention interact, so that each can produce a “drag” on the other—slowing it or
partially diverting it—when they conflict.768

Trying to spot a predator in a complex scene can cause attending, priori to stimuli, to cer-
tain shapes and sizes. Attention, overt or covert, to task-targets figures in forming a priority
map for future saccades, in speeding processing of representation of attributes associated with
the target, and in speeding motor behavior—eye movements, pointing, grasping, avoiding.769

Various types of memory can affect allocation of both exogenous and endogenous
attention. For example, short-term memory involved in priming can bias attention to
speed processing of recently perceived items. Memory of a reward history can yield antici-
pations informed by attention. Long-term memory of a scene gist in which a searched-for
object typically occurs can guide attention. Items held unconsciously in long-term mem-
ory can affect allocation of exogenous attention over a half-year after reward learning.
Like all attentional effects, effects deriving from memory can affect perceptual processing
at various processing stages. Sources of memory effects can be as ordinary as priming or as
grand as years-long memory.770
